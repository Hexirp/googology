バシク行列システムを次のように順序数表記としてみなす。この際、[[ユーザーブログ:Koteitan/バシク行列の数式的定義]]を参照している。また、バシク行列の"展開関数"に関する整礎性を仮定している。

項 <math> \mathrm{T} </math> は <math> \mathbb{N} </math> を要素とする行列全体である。

集合 <math> \mathrm{O} </math> を <math> S \in \mathrm{O} \Longleftrightarrow S = () </math> と定義する。

集合 <math> \mathrm{S} </math> を <math> S \in \mathrm{S} \Longleftrightarrow \forall y \ldotp S _ { ( X - 1 ) y } = 0 </math> と定義する。

集合 <math> \mathrm{L} </math> を <math> S \in \mathrm{L} \Longleftrightarrow S \notin \mathrm{O} \land S \notin \mathrm{S} </math> と定義する。

展開関数 <math> S [ n ] </math> を <math> \boldsymbol{G} \boldsymbol{B} ^ { ( 0 ) } \boldsymbol{B} ^ { ( 1 ) } \boldsymbol{B} ^ { ( 2 ) } \cdots \boldsymbol{B} ^ { ( n ) } </math> と定義する。この操作は <math> S \in \mathrm{L} </math> である場合しか定義されない。

比較関数 <math> < _ \mathrm{T} </math> を行列を列のリストとして列を自然数のリストとした辞書式順序で定義する。

標準形の項 <math> \mathrm{OT} </math> を再帰的に定義する。

# 任意の <math> S </math> と <math> n </math> に対して <math> S = ( \underbrace{ 0, 0, \ldots, 0 } _ { n + 1 } ) ( \underbrace{ 1, 1, \ldots, 1 } _ { n + 1 } ) </math> であるならば <math> S \in \mathrm{OT} </math> である。
# 任意の <math> S </math> と <math> n </math> に対して <math> S \in \mathrm{L} \land S \in \mathrm{OT} </math> であるならば <math> S [ n ] \in \mathrm{OT} </math> である。

ここで <math> \mathrm{OT} </math> が帰納的可算集合であること（半決定可能であること）は容易にわかるが、帰納的集合であること（決定可能であること）は自明ではない。しかし、この集合は帰納的集合であり決定可能である、つまり、帰属を判定するアルゴリズムが存在すると見られ、そのアルゴリズムの候補が存在する。この候補アルゴリズムは [http://gyafun.jp/ln/basmat.cgi Bachicu Matrix Calculator] を参考にしている。

* <math> S \in \mathrm{OT} </math> を以下のように定める。
*# <math> S \in \mathrm{OT} ( ( \underbrace{ 0, 0, \ldots, 0 } _ { Y + 1 } ) ( \underbrace{ 1, 1, \ldots, 1 } _ { Y + 1 } ) ) </math> と同値である。ここで <math> Y </math> は <math> S </math> の列の長さである。
* <math> S \in \mathrm{OT} ( Z ) </math> を以下のように定める。
*# <math> S \in \mathrm{OT} ( Z, 0 ) </math> と同値である。
* <math> S \in \mathrm{OT} ( Z, n ) </math> を以下のように定める。
*# <math> S < _ \mathrm{T} Z [ n ] </math> であるとき <math> S \in \mathrm{OT} ( Z [ n ] ) </math> と同値である。
*# <math> S = _ \mathrm{T} Z [ n ] </math> であるとき <math> \mathrm{True} </math> と同値である。
*# <math> S > _ \mathrm{T} Z [ n ] </math> であるとき、
*## <math> X _ S \le X _ { Z [ n ] } </math> であるとき <math> \mathrm{False} </math> と同値である。ここで <math> X _ S </math> は <math> S </math> の行の長さであり、 <math> X _ { Z [ n ] } </math> は <math> Z [ n ] </math> の行の長さである。
*## <math> X _ S >   X _ { Z [ n ] } </math> であるとき <math> S \in \mathrm{OT} ( Z, n + 1 ) </math> と同値である。ここで <math> X _ S </math> は <math> S </math> の行の長さであり、 <math> X _ { Z [ n ] } </math> は <math> Z [ n ] </math> の行の長さである。

後者関数 <math> \mathrm{pred} ( S ) </math> を <math> \boldsymbol{S} _ 0 \cdots \boldsymbol{S} _ { X - 2 } </math> と定義する。ここで <math> X </math> は <math> S </math> の行の長さである。この操作は <math> S \in \mathrm{S} </math> である場合しか定義されない。

基本列関数 <math> \mathrm{fseq} ( S, n ) </math> を <math> S [ n ] </math> と定義する。この操作は <math> S \in \mathrm{L} </math> である場合しか定義されない。
