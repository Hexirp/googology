Bachmann ψ (recast) は Bachmann ψ として考案された順序数を Rathjen が再構成したものである。そもそも Bachmann ψ は史上初の順序数崩壊関数であったが、その定義は全ての極限順序数の基本列が必要になるなど計算が面倒なものであった。それを Rathjen が再構成 (recast) して計算が簡便になっている。<ref>GWiki 2019 § 3.1</ref>

限界は[[バッハマン・ハワード順序数]] (Bachmann–Howard ordinal) である<ref> GWiki 2019 § 5</ref>。そも、名前が同じ人に由来しているので、逆に限界という意味で順序数の名前がこうなったのかもしれない。

== 定義 ==

定義は次のようになる<ref>GWiki 2019 § 3.1</ref>。ここで <math> \Omega </math> はある一つの不可算基数、例えば最小の不可算順序数 (aleph one) である。

<math>
  \begin{eqnarray*}
    C ^ \Omega ( \alpha, \beta ) & \coloneqq & \mathrm{closure} ( \{ +, \xi \mapsto \omega ^ \xi, \xi _ { < \alpha } \mapsto \psi _ \Omega ( \xi ) \} ) ( \beta \cup \{ 0, \Omega \} ) \\
    \psi _ \Omega ( \alpha ) & \coloneqq & \min \{ \rho < \Omega : C ^ \Omega ( \alpha, \rho ) \cap \Omega = \rho \} \\
  \end{eqnarray*}
</math>

ただし <math> \mathrm{closure} ( S ) ( \alpha ) </math> は、関数群 <math> S </math> に対する <math> \alpha </math> の閉包である。

== 計算 ==

<math> \psi _ \Omega ( 0 ) </math> を計算する。まずは <math> C ^ \Omega ( 0, 0 ) = \{ 0, 1, 2, \omega, \omega ^ \omega, \omega ^ { \omega ^ \omega + \omega }, \Omega, \Omega + 1, \Omega + \omega, \omega ^ { \Omega + 1 }, \omega ^ { \Omega + \Omega }, \omega ^ { \omega ^ { \Omega + \Omega } }, \ldots \} </math> である。さらに <math> C ^ \Omega ( 0, \beta ) </math> は <math> \beta < \varepsilon _ 0 </math> のとき <math> C ^ \Omega ( 0, 0 ) </math> と等しい。加えて <math> C ^ \Omega ( 0, \varepsilon _ 0 ) </math> のときも等しい。

よって <math> C ^ \Omega ( 0, 0 ) \cap \Omega = \varepsilon _ 0 </math> であることから <math> C ^ \Omega ( 0, \varepsilon _ 0 ) \cap \Omega = \varepsilon _ 0 </math> である。自明に <math> \varepsilon _ 0 </math> は <math> C ^ \Omega ( \alpha, \rho ) \cap \Omega = \rho </math> を満たす最小の順序数である。

ゆえに <math> \psi _ \Omega ( 0 ) = \varepsilon _ 0 </math> である。

<math> \psi _ \Omega ( 1 ) </math> を計算する。ここでは <math> \xi _ { < \alpha } \mapsto \psi _ \Omega ( \xi ) </math> が <math> \xi _ { < 1 } \mapsto \psi _ \Omega ( \xi ) </math> となって <math> \psi _ \Omega ( 0 ) = \varepsilon _ 0 </math> が <math> C ^ ( 1, \beta ) </math> に含まれるようになる。

よって <math> C ^ \Omega ( 1, 0 ) = C ^ \Omega ( 1, 1 ) = \cdots = C ^ \Omega ( 1, \varepsilon _ 1 ) = \varepsilon _ 1 </math> になる。ゆえに <math> \psi _ \Omega ( 1 ) = \varepsilon _ 1 </math> である。

<math> \psi _ \Omega ( \omega ) </math> を計算する。まず <math> C ^ \Omega ( \omega, 0 ) = \{ 0, 1, \omega, \omega ^ 2, \omega ^ \omega, \varepsilon _ 0, \varepsilon _ 1, \ldots \} </math> であり <math> C ^ \Omega ( \omega, 0 ) \cup \Omega = \varepsilon _ \omega </math> である。ゆえに <math> \psi _ \Omega ( \omega ) = \varepsilon _ \omega </math> である。

<math> \psi _ \Omega ( \psi _ \Omega ( 0 ) ) = \psi _ \Omega ( \varepsilon _ 0 ) </math> を計算する。 <math> C ^ \Omega ( \varepsilon _ 0, 0 ) = \{ \varepsilon _ 0, \varepsilon _ 1, \varepsilon _ \omega \ldots \} </math> であり <math> C ^ \Omega ( \varepsilon _ 0, 0 ) \cup \Omega = \varepsilon _ { \varepsilon _ 0 } </math> である。ゆえに <math> \psi _ \Omega ( \omega ) = \varepsilon _ \omega </math> である。

このように積み重ねると <math> 0, \psi _ \Omega ( 0 ), \psi _ \Omega ( \psi _ \Omega ( 0 ) ), \ldots \Longrightarrow \zeta _ 0 </math> に達する。

<math> \psi _ \Omega ( \zeta _ 0 ) </math> を計算する。 <math> C ^ \Omega ( \varepsilon _ 0, 0 ) = \{ \varepsilon _ 0, \varepsilon _ { \varepsilon _ 0 }, \varepsilon _ { \varepsilon _ { \varepsilon _ 0 } } \ldots \} </math> であるゆえに <math> \psi _ \Omega ( \zeta _ 0 ) = \zeta _ 0 </math> である。

<math> \psi _ \Omega ( \Omega ) </math> を計算する。このとき <math> \xi _ { < \Omega } \mapsto \psi _ \Omega ( \xi ) </math> であるため <math> \alpha < \zeta _ 0 </math> では許されなかった <math> \psi _ \Omega </math> の無制限のネストが許される。よって <math> \zeta _ 0 </math> と同じような状態になり、ゆえに <math> \psi _ \Omega ( \Omega ) = \zeta _ 0 </math> である。

<math> \psi _ \Omega ( \Omega + 1, 0 ) </math> を計算する。このとき <math> \xi _ { < \Omega + 1 } \mapsto \psi _ \Omega ( \xi ) </math> であるため <math> C ^ \Omega ( \Omega + 1, 0 ) </math> に <math> \psi _ \Omega ( \Omega ) </math> が'''含まれる'''。ゆえに <math> \psi _ \Omega ( \Omega + 1 ) = \varepsilon _ { \zeta _ 0 + 1 } </math> である。そのまま増やした <math> \zeta _ 0, \zeta _ 0 + 1, \zeta _ 0 + 2, \ldots </math> では含まれず、そのせいで <math> \psi _ \Omega ( \zeta _ 0 + 1 ) = \zeta _ 0 </math> になる。

このまま計算を続けていくと以下のようになる。これは Madore ψ に一致している。

<math>
  \begin{eqnarray*}
    \psi _ \Omega ( 0 ) & = & \varepsilon _ 0 \\
    \psi _ \Omega ( 1 ) & = & \varepsilon _ 1 \\
    \psi _ \Omega ( \omega ) & = & \varepsilon _ \omega \\
    \psi _ \Omega ( \psi _ \Omega ( 0 ) ) & = & \varepsilon _ { \varepsilon _ 0 } \\
    \psi _ \Omega ( \zeta _ 0 ) & = & \zeta _ 0 \\
    \psi _ \Omega ( \Omega ) & = & \zeta _ 0 \\
    \psi _ \Omega ( \Omega + 1 ) & = & \varepsilon _ { \zeta _ 0 + 1 } \\
    \psi _ \Omega ( \Omega + \omega ) & = & \varepsilon _ { \zeta _ 0 + \omega } \\
    \psi _ \Omega ( \Omega + \psi _ \Omega ( \Omega ) ) & = & \varepsilon _ { \zeta _ 0 + \zeta _ 0 } \\
    \psi _ \Omega ( \Omega + \Omega ) & = & \zeta _ 1 \\
  \end{eqnarray*}
</math>

== 順序数表記 ==

Bachmann ψ (recasted) を元にした順序数表記は探しても見つからなかった。ここのは私が独自に定めたものである。この順序数表記は良く知られた順序数表記である Buchholz ψ と違って <math> \omega ^ \alpha </math> が基本記号（表記を構成する様々なアリティの定数記号および関数記号）に含まれている。そのため、この順序数表記が正しいのかはまだわかってない。

=== 定義 ===

集合 <math> T </math> を次のような条件を満たす最小の集合であるとして定める。

# <math> 0 \in T </math> である。
# <math> a \in T </math> に対して <math> \omega ^ a \in T </math> である。
# <math> a \in T </math> に対して <math> \psi _ \Omega ( a ) \in T </math> である。
# <math> \Omega \in T </math> である。
# <math> 1 \lt k \le \omega </math> を取り、単項（条件 2, 3 で構成された要素）の列 <math> a _ 0, \ldots, a _ k \in T </math> に対して <math> a _ 0 + \cdots + a _ k \in T </math> である。

集合 <math> T </math> の上の狭義全順序 <math> < </math> を次のように帰納的に定める。

# <math> 0 < 0 </math> ではない。
# <math> 0 </math> ではない <math> a \in T </math> に対して <math> 0 < a </math> である。
# <math> 0 </math> ではない <math> a \in T </math> に対して <math> a < 0 </math> ではない。
# いずれも <math> 0 </math> ではない <math> a, b \in T </math> に対して、
## <math> a, b </math> の双方とも単項であるとき、
### <math> a = \omega ^ { a' }, \, b = \omega ^ { b' } </math> のとき <math> a' < b' </math> と同値である。
### <math> a = \omega ^ { a' }, \, b = \psi _ \Omega ( b' ) </math> のとき <math> a' < \psi _ \Omega ( b' ) </math> と同値である<ref group="注釈"><math> \psi _ \Omega ( b' ) </math> は必ず <math> \xi \mapsto \omega ^ \xi </math> の不動点である</ref>。
### <math> a = \omega ^ { a' }, \, b = \Omega </math> の時 <math> a' < \Omega </math> と同値である<ref group="注釈"><math> \Omega </math> は最大元ではない</ref>。
### <math> a = \psi _ \Omega ( a' ), \, b = \omega ^ { b' } </math> のとき <math> \psi _ \Omega ( a' ) < b' </math> と同値である。
### <math> a = \psi _ \Omega ( a' ), \, b = \psi _ \Omega ( b' ) </math> のとき <math> a' < b' </math> と同値である。
### <math> a = \psi _ \Omega ( a' ), \, b = \Omega </math> のとき <math> a < b </math> である。
### <math> a = \Omega, \, b = \omega ^ { b' } </math> のとき <math> a < b' </math> と同値である。
### <math> a = \Omega, \, b = \psi _ \Omega ( b' ) </math> のとき <math> a < b </math> ではない。
### <math> a = \Omega, \, b = \Omega </math> のとき <math> a < b </math> ではない。
## <math> a </math> が単項で <math> b </math> が単項ではないとき、
### <math> b = { b' } _ 0 + \cdots + { b' } _ k </math> のとき、
#### <math> a < { b' } _ 0 </math> であるならば <math> a < b </math> である。
#### <math> a = { b' } _ 0 </math> であるならば <math> a < b </math> である。
#### <math> a > { b' } _ 0 </math> であるならば <math> a < b </math> ではない。
## <math> a </math> が単項ではなく <math> b </math> が単項のとき、
### <math> a = { a' } _ 0 + \cdots + { a' } _ k </math> のとき、
#### <math> { a' } _ 0 < b </math> であるならば <math> a < b </math> である。
#### <math> { a' } _ 0 = b </math> であるならば <math> a < b </math> ではない。
#### <math> { a' } _ 0 > b </math> であるならば <math> a < b </math> ではない。
## <math> a, b </math> が双方とも単項ではないとき、
### <math> a = { a' } _ 0 + \cdots + { a' } _ m, \, b = { b' } _ 0 + \cdots + { b' } _ n </math> のとき、以下のいずれかが成り立てば <math> a < b </math> である
#### <math> m < n </math> であり、どの <math> i \le m </math> に対しても <math> { a' } _ i = { b' } _ i </math> である。
#### <math> k \le \min \{ m, n \} </math> が存在して <math> a _ k < b _ k </math> を満たし、どの <math> i < k </math> に対しても <math> { a' } _ i = { b' } _ i </math> である。

集合 <math> G ( a ) </math> を次のような条件を満たす最小の集合であるとして定める。

# <math> a = 0 </math> のとき <math> G ( a ) = \emptyset </math> である。
# <math> a = \omega ^ { a' } </math> のとき  <math> G ( a ) = \{ a' \} \cup G ( a' ) </math> である<ref group="注釈">この部分は <math> G ( a ) = G ( a' ) </math> と簡略化できるかもしれない</ref>。
# <math> a = \psi _ \Omega ( a' ) </math> のとき <math> G ( a ) = \{ a' \} \cup G ( a' ) </math> である。
# <math> a = \Omega </math> のとき <math> G ( a ) = \emptyset </math> である。
# <math> a = { a' } _ 0 + \cdots + { a' } _ k </math> のとき <math> G ( a ) = G ( { a' } _ 0 ) \cup \cdots \cup G ( { a' } _ k ) </math> である。

集合 <math> OT </math> を次のように帰納的に定める。

# <math> 0 \in OT </math> である。
# <math> a \in OT </math> に対して <math> a = 0 </math> か <math> a = { a' } _ 0 + \cdots + { a' } _ k </math> であれば <math> \omega ^ a \in OT </math> である。
# <math> a \in OT </math> に対して <math> \forall a', \, a' \in G ( a ) \to a' < a </math> であれば <math> \psi _ \Omega ( a' ) \in OT </math> である。
# <math> \Omega \in OT </math> である。
# <math> a _ 0, \ldots, a _ k \in OT </math> に対して <math> a _ 0 \ge \cdots \ge a _ k </math> であれば <math> a _ 0 + \cdots + a _ k \in OT </math> である。

対応 <math> o ( a ) </math> を次のようにして定める。

# <math> a = 0 </math> のとき <math> o ( a ) = 0 </math> である。
# <math> a = \omega ^ { a' } </math> のとき <math> o ( a ) = \omega ^ { o ( a' ) } </math> である。
# <math> a = \psi _ \Omega ( a' ) </math> のとき <math> o ( a ) = \psi _ \Omega ( o ( a' ) ) </math> である。
# <math> a = \Omega </math> のとき <math> o ( a ) = \Omega </math> である。
# <math> 1 \lt k \le \omega </math> を取って <math> a = { a' } _ 0 + \cdots + { a' } _ k </math> のとき <math> o ( a ) = o ( { a' } _ 0 ) + \cdots + o ( { a' } _ k ) </math> である。

=== 計算 ===

標準形の判定はこのようになる。

<math>
  \begin{array}{ll}
    0 & T \\
    \psi _ \Omega ( 0 ) & T \\
    \psi _ \Omega ( \Omega ) & T \\
    \omega ^ 0 & T \\
    \omega ^ 0 + \omega ^ 0 & T \\
    \psi _ \Omega ( 0 ) + \psi _ \Omega ( 0 ) & T \\
    \omega ^ { \psi _ \Omega ( 0 ) } & F \\
  \end{array}
</math>

=== 性質 ===

このような性質が期待される。証明はまだない。

* <math> o \colon OT \to \mathrm{On} </math> は <math> OT </math> の上の <math> < </math> から通常の <math> < </math> へ順序を保つ写像である。このことから <math> OT </math> の上の <math> < </math> は整列順序を構成する。

一番右に一番奥にある要素を求める。 ω だったら並べていくだけ。 Buchholz ψ と違って ψ 関数は必ずネストへ展開される。カントールの標準形の展開が組み込み。要素が Ω の時だけネストになる。ネストされる範囲は親を辿って行って最初に見つかった ψ までか？

<math> a \in OT </math> の基本列は次のようになる。

# <math> a = 0 </math> のとき <math> \mathrm{cf} ( a ) = 0 </math> であり、基本列は{{wja|空関数}}により与えられる。
# <math> a = \omega ^ b </math> のとき
## <math> b = 0 </math> のとき <math> \mathrm{cf} ( a ) = 1 </math> であり、基本列は <math> a [ 0 ] = 0 </math> により与えられる。
## <math> b = c + \omega ^ 0 </math> のとき <math> \mathrm{cf} ( a ) = \omega </math> であり、基本列は <math> a [ n ] = \omega ^ c \times n </math> により与えられる<ref group="注釈">ここでの乗算は略記と捉えてほしい</math>。
## <math> b \in L </math> のとき <math> \mathrm{cf} ( a ) = \mathrm{cf} ( b ) </math> であり、基本列は <math> a [ n ] = \omega ^ { b [ n ] } </math> により与えられる。
# <math> a = \psi _ \Omega ( b ) </math> のとき
## <math> b = 0 </math> のとき <math> \mathrm{cf} ( a ) = \omega </math> であり、基本列は <math> a [ 0 ] = 0, \, a [ n + 1 ] = \omega ^ { a [ n ] } </math> により与えられる。
## <math> b = c + \omega ^ 0 </math> のとき <math> \mathrm{cf} ( a ) = \omega </math> であり、基本列は <math> a [ 0 ] = \psi _ \Omega ( c ) + \omega ^ 0, \, a [ n + 1 ] = \omega ^ { a [ m ] } </math> により与えられる。
## <math> \mathrm{cf} ( b ) = \omega </math> のとき <math> \mathrm{cf} ( a ) = \omega </math> であり、基本列は <math> a [ n ] = \psi _ \Omega ( b [ n ] ) </math> により与えられる。
## <math> \mathrm{cf} ( b ) = \Omega </math> のとき <math> \mathrm{cf} ( a ) = \omega </math> であり、基本列は <math> a [ n ] = \psi _ \Omega ( b [ c [ n ] ] ) </math> と <math> c [ 0 ] = 0, \, c [ n + 1 ] = \psi _ \Omega ( b [ c [ n ] ] ) </math> により与えられる。
# <math> a = \Omega </math> のとき <math> \mathrm{cf} ( a ) = \Omega </math> であり、基本列は <math> a [ n ] = n </math> により与えられる。
# <math> a = a _ 0 + \cdots + a _ k </math> のとき <math> \mathrm{cf} ( a ) = \mathrm{cf} ( a _ k ) </math> であり、基本列は <math> a [ n ] = a _ 0 + \cdots + ( a _ k [ n ] ) </math> により与えられる。
